# DevFlow — Developer Focus & Task Manager

DevFlow is a lightweight productivity tool built for developers who need to stay focused, manage tasks, and track their daily progress — all in one place. No sign-up, no backend, no distractions.

## Features

- **Task Board** — Add, complete, and delete tasks with Low, Medium, and High priority tags
- **Priority Filtering** — Filter your task board by priority level
- **Pomodoro Timer** — 25-minute work sessions with 5-minute breaks and sound notification
- **Current Task Display** — Timer shows which task you are currently focusing on
- **Daily Log** — Completed tasks are automatically logged with timestamps
- **Focus Time Tracking** — Tracks total focus minutes accumulated in a day
- **Session Streak Counter** — Counts how many pomodoro sessions you complete
- **Dark/Light Mode** — Toggle between themes, preference saved across sessions
- **Reset Day** — Clear your log, focus time and streak to start fresh
- **Persistent Storage** — All data saved locally, survives page refreshes

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Media Queries)
- Vanilla JavaScript (ES6)
- localStorage API
- Web Audio API

## How to Use

1. Add a task using the Task Board input and select a priority
2. Click Start on the Focus Timer to begin a 25-minute work session
3. Complete tasks as you finish them — they move to the Daily Log
4. Use the filter buttons to view tasks by priority
5. Click Reset Day to clear your daily stats and start fresh
